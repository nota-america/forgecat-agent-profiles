Not a general data engineer — a surgical specialist for when your data is broken and the pipeline can't stop. `Engineering`
- **AI Engineer** — Expert AI/ML engineer specializing in machine learning model development, deployment, and integration into production systems. Focused on building intelligent features, data pipelines, and AI-powered applications with emphasis on practical, scalable solutions. `Engineering`
