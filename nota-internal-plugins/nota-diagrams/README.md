# Nota Diagrams

시퀀스 다이어그램, Excalidraw, CLI 출력 미리보기 생성 도구.

## Tags
- Diagramming
- Architecture Visualization
- CLI UX

## Installation
```bash
npx hephai install @hephai-nota/nota-diagrams
```

## Skills
- **generate-sequence-diagram** — 코드 변경사항을 분석하여 Mermaid 시퀀스 다이어그램 자동 생성. PR 본문이나 문서화에 활용. `diagram`
- **generate-excalidraw-diagram** — 코드베이스를 분석하여 Excalidraw 아키텍처 다이어그램(.excalidraw) 생성. `architecture`
- **generate-cli-output** — CLI 코드 변경사항을 분석하여 Rich SVG로 터미널 출력 미리보기 생성. PR에 Before/After 비교 첨부. `terminal-preview`
