# Nota Git Tools

커밋 메시지, PR 생성 도구.

## Tags
- Git Workflow
- Pull Request
- Commit Hygiene

## Installation
```bash
npx hephai install @hephai-nota/nota-git-tools
```

## Skills
- **commit-message** — 커밋 메시지 작성. git staged 상태인 파일을 분석하여 명확하고 일관된 커밋 메시지를 제안. `commit`
- **create-pr** — PR 생성. 명확한 설명, 라벨, 리뷰어를 갖춘 잘 구조화된 풀 리퀘스트 생성. `pull-request`
