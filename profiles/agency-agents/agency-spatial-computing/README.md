# Agency Spatial Computing

Spatial Computing division from The Agency — 6 specialized agent personalities from msitarzewski/agency-agents. Each agent has deep domain expertise, unique personality, and production-ready workflows.

## Installation
```bash
npx forgecat install @forgecat/agency-spatial-computing
```

## Agents
- **macOS Spatial/Metal Engineer** — Native Swift and Metal specialist building high-performance 3D rendering systems and spatial computing experiences for macOS and Vision Pro `Spatial Computing`
- **Terminal Integration Specialist** — Terminal emulation, text rendering optimization, and SwiftTerm integration for modern Swift applications `Spatial Computing`
- **visionOS Spatial Engineer** — Native visionOS spatial computing, SwiftUI volumetric interfaces, and Liquid Glass design implementation `Spatial Computing`
- **XR Cockpit Interaction Specialist** — Specialist in designing and developing immersive cockpit-based control systems for XR environments `Spatial Computing`
- **XR Immersive Developer** — Expert WebXR and immersive technology developer with specialization in browser-based AR/VR/XR applications `Spatial Computing`
- **XR Interface Architect** — Spatial interaction designer and interface strategist for immersive AR/VR/XR environments `Spatial Computing`
