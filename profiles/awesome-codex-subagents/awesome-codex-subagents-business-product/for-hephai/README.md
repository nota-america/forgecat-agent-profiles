# 08. Business & Product

Support agents for requirements, UX, and engineering-adjacent writing tasks.

## Tags
- Business & Product

## Installation
```bash
npx forgecat install @hephai-nota/awesome-codex-subagents-business-product
```

## Agents
- **business-analyst** — Use when a task needs requirements clarified, scope normalized, or acceptance criteria extracted from messy inputs before engineering work starts. `gpt-5.4` `Business & Product`
- **content-marketer** — Use when a task needs product-adjacent content strategy or messaging that still has to stay grounded in real technical capabilities. `gpt-5.3-codex-spark` `Business & Product`
- **customer-success-manager** — Use when a task needs support-pattern synthesis, adoption risk analysis, or customer-facing operational guidance from engineering context. `gpt-5.3-codex-spark` `Business & Product`
- **legal-advisor** — Use when a task needs legal-risk spotting in product or engineering behavior, especially around terms, data handling, or externally visible commitments. `gpt-5.4` `Business & Product`
- **product-manager** — Use when a task needs product framing, prioritization, or feature-shaping based on engineering reality and user impact. `gpt-5.4` `Business & Product`
- **project-manager** — Use when a task needs dependency mapping, milestone planning, sequencing, or delivery-risk coordination across multiple workstreams. `gpt-5.4` `Business & Product`
- **sales-engineer** — Use when a task needs technically accurate solution positioning, customer-question handling, or implementation tradeoff explanation for pre-sales contexts. `gpt-5.3-codex-spark` `Business & Product`
- **scrum-master** — Use when a task needs process facilitation, iteration planning, or workflow friction analysis for an engineering team. `gpt-5.3-codex-spark` `Business & Product`
- **technical-writer** — Use when a task needs release notes, migration notes, onboarding material, or developer-facing prose derived from real code changes. `gpt-5.3-codex-spark` `Business & Product`
- **ux-researcher** — Use when a task needs UI feedback synthesized into actionable product and implementation guidance. `gpt-5.4` `Business & Product`
- **wordpress-master** — Use when a task needs WordPress-specific implementation or debugging across themes, plugins, content architecture, or operational site behavior. `gpt-5.4` `Business & Product`
